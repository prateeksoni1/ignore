### Installation
```bash
  npx creact-react-app tutorial
```

### Alternate package manager
- yarn
- pnpm

### To start app, run
```bash
  npm start
  npm run start
  yarn start
```

## Hooks
- useState
  It is used to create states that re-render the app on change
- useEffect
  It is used to run a function whenever anything in deps array change
